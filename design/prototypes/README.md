# TrustDB UI Modernization — Prototype Set

This folder contains the first visual direction for a unified TrustDB interface system:

- `generated/trustdb-client-ui-v1.png` — desktop client dashboard
- `generated/trustdb-web-admin-ui-v1.png` — web operations console
- `generated/trustdb-official-website-ui-v1.png` — official marketing website
- `generated/trustdb-official-website-ui-v2.png` — lower-density official website revision

Core visual rules: preserve acid green `#00FF22`, carbon black, warm off-white, proof-chain semantics, data-topography fields, oversized editorial typography, and a continuous “proof signal” that becomes the primary GSAP motion motif during implementation.

Reference captures used for the exploration are kept under `references/`.

The revised official-site layout and motion specification are documented in `OFFICIAL_WEBSITE_V2.md`.
